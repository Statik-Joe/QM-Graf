Kräfteaddition & Vektoren

Verändere die Winkel und Beträge der Kräfte, um die Resultierende (grün) zu ermitteln.

Kraft F₁ (Blau)

Kraft F₂ (Rot)

Analytische Addition (Vektorzerlegung)

F₁ Zerlegung:

F₁ₓ = F₁ · cos(α₁) = 0 N

F₁ᵧ = F₁ · sin(α₁) = 0 N

F₂ Zerlegung:

F₂ₓ = F₂ · cos(α₂) = 0 N

F₂ᵧ = F₂ · sin(α₂) = 0 N

Resultierende Gesamtkraft F_R (Grün):

F_Rₓ = F₁ₓ + F₂ₓ = 0 N

F_Rᵧ = F₁ᵧ + F₂ᵧ = 0 N

F_R = √(F_Rₓ² + F_Rᵧ²) = 0 N

Wirkungswinkel φ = 0°
